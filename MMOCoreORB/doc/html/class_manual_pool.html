bool ManualPool::update ( )
privatevirtual

The update function checks the ResourceSpawn items in spawnedResources to see if they have expired. If they have not expired, no action is taken, but if they have expired, we replace them according to the rules.

The manual pool doesn't spawn anything on it's own it just removes anything that happens to be expired
