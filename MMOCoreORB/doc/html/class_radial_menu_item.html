Detailed Description

Please see Radial in the wiki for a list of client side radial id's. Also, please note that radial id's are handled in RadialManager.cpp and most have already been defined. Try to find one suitable to your purpose, or piggy back the SERVER_MENUX ones which Call an objects associated virtual method.

Callback Types known: 0x01 = Default 0x03 = Callback
